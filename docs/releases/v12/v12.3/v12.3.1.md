This is a new patch release of the `github-actions` package. It includes small, non-breaking changes and should require no refactoring. Please read the description of changes below.
<!-- alex-c-line-release-summary-end -->

## Description of Changes

<!-- user-editable-section-start -->
- Remove manual installation of NPM in `npm-registry-publish`
  - Now that PNPM owns the entire publish step with its publish command rather than delegating to NPM, we shouldn't need to install NPM at some specific version ourselves. PNPM should just already be compatible with OIDC in v11.
