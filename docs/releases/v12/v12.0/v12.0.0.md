**Status**: Released

This is a new major release of the `github-actions` package. It has the potential to introduce breaking changes that may require a large amount of refactoring. Please read the description of changes and migration notes below for more information.
<!-- alex-c-line-release-summary-end -->

## Description of Changes

<!-- user-editable-section-start -->
- Replace `comment-outdated` workflow with a more full `audit-javascript-dependencies` workflow.
  - This does a more full audit of the dependencies using `alex-c-line`'s `dependency-audit` command.
  - See [alex-c-line v2.9.0](https://github.com/alextheman231/alex-c-line/blob/main/docs/releases/v2/v2.9/v2.9.0.md) for more information.

## Migration Notes

Replace

```yml
uses: alextheman231/github-actions/.github/workflows/comment-outdated.yml
with:
  pull-request-number: ${{ github.event.pull_request.number }}
secrets:
  APP_ID: ${{ vars.ALEX_UP_BOT_APP_ID }}
  APP_PRIVATE_KEY: ${{ secrets.ALEX_UP_BOT_PRIVATE_KEY }}
```

with

```yml
uses: alextheman231/github-actions/.github/workflows/audit-javascript-dependencies.yml
with:
  pull-request-number: ${{ github.event.pull_request.number }}
  APP_ID: ${{ vars.ALEX_UP_BOT_APP_ID }}
secrets:
  APP_PRIVATE_KEY: ${{ secrets.ALEX_UP_BOT_PRIVATE_KEY }}
```
