## Description of Changes

<!-- user-editable-section-start -->
- Remove the branch restriction on audit-javascript-dependencies in favour of basing it off of lockfile change
  - This ensures we get visibility into the current state of dependencies any time the dependencies get changed.
  - The lockfile is generally the best indicator of this so we can run the workflow based on changes to that.
