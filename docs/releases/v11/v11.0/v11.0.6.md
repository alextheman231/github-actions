## Description of Changes

<!-- user-editable-section-start -->
- Add the `--save-exact` flag to the Python updates.
    - Previously it would've rewritten the version ranges using `>=` even if we initially used `==`.
    - This changes that so that we always rewrite the versions using `==`, corresponding to an exact pin.
- Format the `pyproject.toml` file after updates.
    - `pdm` keeps trying to fight `pyproject-fmt`...
