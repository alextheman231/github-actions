## Description of Changes

<!-- user-editable-section-start -->
- Add a step to apply formatting to update-dependencies
    - This may then reduce the amount of manual changes needed after updates, as formatting errors will also be applied beforehand too.
    - This is especially useful throughout my organisation, especially if I end up updating the ESLint plugin, for example.
    - Formatting application also comes through on its own commit, so we don't lose visibility of it.
